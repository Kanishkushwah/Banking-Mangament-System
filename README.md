# Banking Management System

A Java-based Banking Management System developed using Object-Oriented Programming (OOP) concepts. This project simulates basic banking operations such as account creation, login authentication, deposits, withdrawals, fund transfers, and transaction management.

## Features

- User Login Authentication
- Create Customer Accounts
- Savings Account Management
- Deposit Money
- Withdraw Money
- Fund Transfer Between Accounts
- Transaction History
- Balance Inquiry
- Exception Handling for Insufficient Balance
- Modular OOP-Based Design
